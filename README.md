# Astro UNOAuth Template

- UnoCSS
- Preact (compat + signals)
- Million Compiler v2
- Nanostores
- Lucia Auth (+ Oauth)
- Astro DB
- Vercel
- Sentry (with spotlight.js)
- StyleLint + ESLint
